# WP-Circle-Slider
This is an Owl Carousel rounded slider shortcode plugin for WordPress.

<h2>Shortocode</h2>

[cs_slider_shortcode]

  loop: false <br>

  touchDrag: false <br>
  mouseDrag: false <br>
  pullDrag: false <br>
  freeDrag: false <br>
  smartSpeed: 450 <br>
 <b>NOTE: These options are not extendable yet. </b>
